    timeline(*attrs) click to toggle source

    To track your model changes call timeline in any Active Record model class.

    class Product < ActiveRecord::Base
  timeline
end

    To track only one or a set of attributes(columns) call timeline :attr1, :attr2 .

    class Product < ActiveRecord::Base
  timeline :price, :description
end
